## Chia code theo 4 tuần

- `week1-home-products.ts`: Trang chủ, danh sách sản phẩm, chi tiết sản phẩm
- `week2-cart-checkout.ts`: Giỏ hàng và thanh toán
- `week3-auth.ts`: Đăng nhập / đăng ký và điều kiện truy cập giỏ hàng
- `week4-ai-chatbot.ts`: Widget AI chatbot gợi ý mua hàng
- `index.ts`: Tổng hợp cả 4 mốc tuần

Các file này dùng để báo cáo tiến độ và chỉ rõ phần nào của dự án ứng với từng tuần.
